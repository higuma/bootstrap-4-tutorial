Figures

Image with a caption

Use a figure and add a caption text with figcaption .

For a caption below an image

  • figure.figure
    • img.figure-img.img-fluid
    • figcaption.figure-caption

For a caption above an image

  • figure.figure
    • figcaption.figure-caption.mb-2
    • img.figure-img.img-fluid

Caption placement

Set .text-(left|center|right) to figcaption (⇒ Text utility )

figcaption.figure-caption.text-left

figcaption.figure-caption.text-center

figcaption.figure-caption.text-right
