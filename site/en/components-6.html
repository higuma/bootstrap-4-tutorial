Input group

The input group is the extension for input elements (input, textarea, etc.) to add text, buttons, or button groups.

Basic example

Prepending
@
Structure
  • Input group (outer frame) div.input-group
    • Prepending group div.input-group-prepend
      • span.input-group-text Attribute
        • (Accessibility) attribute id="basic-addon1"
        • Text @
    • Form control input.form-control Attributes
      • Attribute type="text"
      • Attribute placeholder="Username"
      • Accessibility attribute aria-label="Username"
      • Accessibility attribute aria-describedby="basic-addon1"

Bellow are guidelines on using input groups (according to the official document).

  • Only one form control can be used in the input group
    • Multiple form controls in a single input group causes problems on Bootstrap style validation
  • Labels must be used outside the input group
    • Or it may cause a problem on accessibility
Caution: The rules are not strict. Some examples in the official documant do not follow above rules.
(On version 4.0) Support for validation is (slightly) incomplete. On displaying validation response, borders are not rounded (see validation example below).
