Lists (ul, ol)

Unstyled

Description list alignment

Elements for the description list (dl , dt , and dd) can be arranged responsively using grid. The following example show how to use grid for building a responsive description list (changed breakpoint from sm to md from the official example).

  • Description list dl.row
    • Description term dt.col-md-3 (placed on left 3/12 on ≥md)
      • Add .text-truncate to truncate text
    • Description details dd.col-md-9 (placed on right 9/12 on ≥md)
    • (Optional) nested description list dl.row (inside is the same way as above)
