List group

Basic example

  • Cras justo odio
  • Dapibus ac facilisis in
  • Morbi leo risus
  • Porta ac consectetur ac
  • Vestibulum at eros
Structure
  • List group ul.list-group Attribute
    • Attribute (sets width) style="width: 320px;"
    • Item li.list-group-item
      • Text (e.g.) Cras justo odio
As default, ul.list-group sets a width to 100% inside a parent. You can use the same techniques as setting sizes of cards (see Cards - Sizing).

Links and buttons

An interactive list group can be built with a parent div.list-group and items (a|button).list-group-item

Structure
  • List group div.list-group
    • Item (a|button).list-group-item Attribute
      • Add .list-group-item-action to respond to hover

Example for a : a.list-group-item.list-group-item-action(.active)

Example for button : button.list-group-item.list-group-item-action(.active)

Contextual classes

Add .list-group-item-(primary|secondary|success|...) to items.

  • (default) Dapibus ac facilisis in
  • .list-group-item-primary This is a primary list group item
  • .list-group-item-secondary This is a secondary list group item
  • .list-group-item-success This is a success list group item
  • .list-group-item-danger This is a danger list group item
  • .list-group-item-warning This is a warning list group item
  • .list-group-item-info This is a info list group item
  • .list-group-item-light This is a light list group item
  • .list-group-item-dark This is a dark list group item

Structure
  • ul.list-group Attribute
    • Sets width style="width: 320px;"
    • Items li.list-group-item Attributes
      • Uses flex .d-flex (see Flex)
      • Horizontal alignment .justify-content-between
      • Vertical alignment .align-items-center
      • Text (e.g.) Cras justo odio
      • Pill badge span.badge.badge-primary.badge-pill
        • Text (e.g.) 14

    Structure
    • List group div.list-group Attribute
      • Accessibility attribute role="tablist"
      • Item a.list-group-item(.list-group-item-action)? Attributes
        • Set .active to an initially-visible item
        • Attribute id="{item-id}"
        • Attribute data-toggle="list"
        • Attribute href="#{content-id}"
        • Accessibility attribute role="tab"
        • Accessibility attribute aria-controls="{content-id}"
    • Content-side div.tab-content
      • Content item div.tab-pane(.fade) Attributes
        • Add.fade for animation (fade in/out) effect
        • Set .show.active to an initially-visible item
        • Attribute id="{content-id}"
        • Accessibility attribute role="tabpanel"
        • Accessibility attribute aria-labelledby="{item-id}"
