      • Text (e.g.) Left

Button toolbar

A button toolbar is used to gather button groups, input groups, form controls, etc.

Structure
  • Button toolbar div.btn-toolbar Attributes
    • Accessibility attribute role="toolbar"
    • Accessibility attribute (e.g.) aria-label="Toolbar with button groups"
    • Button group(s) div.btn-group
      • See Basic example
    • Or also: Input group (⇒ Input group ), etc.
