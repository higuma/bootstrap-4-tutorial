Carousel

Basic (auto-slide only)

Structure
  • Carousel (outer) div.carousel.slide Attribute
    • Attribute (auto-slide) data-ride="carousel" (remove to slide by controls only)
    • Carousel (inner) div.carousel-inner
      • Items div.carousel-item (×3) Attributes
        • (One item only) add .active to the initially-visible item
        • (Optional) interval can be set with data-interval="{msec}".
        • Image img.d-block.w-100 Attribute
          • Attribute src="..."

Collapse

You can toggle collapse (hidden) and expanded (shown) states of a target element with a button.

Example

Live examples below (a on left, button on right, both switch the same target div.collapse#ex-collapse).

a.btn.btn-primary

div.collaps#ex-collapseAnim pariatur cliche reprehenderit, enim eiusmod high life accusamus terry richardson ad squid. Nihil anim keffiyeh helvetica, craft beer labore wes anderson cred nesciunt sapiente ea proident.

Structure
  • Button: one of button or a
    • With a ⇒ a.btn.btn-(primary|secondary|...) Attributes
      • Attribute data-toggle="collapse"
      • Attribute href="#{target-id}" (or an arbitrary CSS selector)
      • Accessibility attribute role="button"
    • With button ⇒ button.btn.btn-(primary|secondary|...) Attributes
      • Attribute type="button"
      • Attribute data-toggle="collapse"
      • Attribute data-target="#{target-id}" (or an arbitrary CSS selector)
    • Common accessibility attributes Attributes
      • aria-expanded="false" (or "true" for initially-shown)
      • aria-controls="{target-id}"
  • Target (can be any element) Attributes
    • Set class .collapse (hidden as default)
    • Add .show to set initially-shown
    • Set a target selector
      • (e.g.) id="{target-id}"
      • Or any CSS selector you need
and buttons above are also made with the collapse function.

Active/disabled items

Add .active to a item to display as active state (white text on primary/blue background).

Regular linkActive linkAnother link

Add .disabled to an item to display as disabled (grayed text).

Regular linkDisabled linkAnother link

However, both classes have no behavior effect. An item with .active does not change when a user select another item. Also, an item with .disabled can be selected. Furthermore, it links to the target if its element is a (not disabled). You can check these behaviors with above examples.

(Advanced) JavaScript behaviors

JavaScript is required for appropriate behaviors. See the example below.

Item 1Item 2Item 3Item 4Item 5 (disabled)
Item 1 is currently active.
Structure (summary) JavaScript code
  • A dropdown button is same as above
  • Sets ID to an item list id="active-disabled-example"
  • Sets .active to Item 1
  • Sets .disabled to Item 5 (disabled)
  • Outside a dropdown, adds span#active-indicator for displaying a currently active item
// Dropdown: Active/disabled states demo
$('#active-disabled-example').delegate('a', 'click', function(event) {
  var item = $(this);
  event.preventDefault();                   // prevent default link behaviors
  if (item.hasClass('disabled')) return;    // implement disabled behaviors
  // update the active item
  $('#active-disabled-example a').removeClass('active');
  item.addClass('active');
  $('#active-indicator').text(item.text());
});
  • Sets click event handler to a parent (item list)
    • Delegates events to descendent items a (more efficient than children receive directly)
  • In an event handler
    • event.preventDefault() prevents an item's defaut behavior.
      • Required if an item is a (or it jumps to the target[href] on click)
      • Not required if an item is button
    • Implements a disabled behavior
      • Checks if an clicked item sets .disabled and just returns (does nothig) if sets
    • Implements an active behavior
      • Removes .active from all items first
      • Adds .active to a clicked item
    • Updates an active item indicator

Menu alignment

A dropdown menu aligns left to a parent (same as a trigger button) as default. Adding .dropdown-menu-right to an item list, it alignes right at a parent (trigger button)'s right side.

Dropdown options

You can specify menu position offset with data-offset="{x-offset-px},{y-offset-px}".

You can change menu position reference to the parent with data-reference="parent". It is useful for a split button to show menu from its parent frame.
