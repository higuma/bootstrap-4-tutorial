Scrollspy

Scrollspy is a functional component which synchronizes the active item display of the indicator (navigation or list group ) components and the horizontal scroll position of the content.

How it works

Scrollspy watches the scrolling content and update the position indicator (navigation or list group) with JavaScript. Requirements are as follows.

  • The indicator component must be Nav or List group
  • The root element of scrolling content requires:
    • position: relative;
    • overflow-y: scroll;
    • Have a height set
  • If the root is body , it satisfies above all requirements

How to use it

  • Use body as the scrolling root, or set attributes to satisfy above requirements
  • Set the following attributes to the scrolling root element
    • data-spy="scroll"
    • data-target="#{indicator-id}"
    • (Optional) data-offset="{pixels}" (scroll calculation offset, default="10" )
  • Inside the scrolling root, set id="{target-id}" to each scrolling target elements
  • The indicator component must be one of followings
    • Navigation
    • List group
  • Set id="{indicator-id}"
  • Set anchors a with href="#{target-id}" into the correcponding indicator items

Toasts

Toasts are used for short-time alert messages. To activate them, use JavaScript code as below.

$('.toast').toast('show');

Normally, it hides automatically after the short moment (500ms as default). You can change the delay time (or keep showing) with Data options.

Data options

Option attributes as below can be used for a toast frame div.toast.

  • data-delay="{delay in milliseconds}" sets delay time to hide a toast ("500"[msec] as default).
  • data-animation="(true|false)" sets CSS animation ("true" as default).
  • data-autohide="(true|false)" hides a toast automatically ("true" as default).

Tooltips

Bootstrap tooltip is customization of tooltip display by title attribute. It utilizes Popper.js for implementation.

  • Browser default tooltip ⇒ Hover here
  • Bootstrap tooltip ⇒ Hover here
Elements which accept key focus (as a , button , etc.) also show tooltips on setting focus by Tab keys.

Initialization

Bootstrap tooltips are inactivated as defaut. JavaScript is required to activate. Below is a simple code to activate all tooltips.

$('[data-toggle="tooltip"]').tooltip();
Caution: jQuery, Popper.js, and bootstrap.js are required. Use this script after those libraries.

How to use it

  • Available for any elements with setting following attributes
    • data-toggle="tooltip"
    • title="{tooltip-text}"

Example ⇒ Hover here

Directions

Set attribute data-placement=("top"|"right"|"bottom"|"left") ( "top" as default)

Example

Tooltips always watch viewport size and elements' posistions, and switch to the opposite sides when space is not enough. It can be demonstrated with the above example by scrolling (top/bottom) or changing viewport width (left/right).

Using HTML

  • Add attribute data-html="true"
  • Set text with HTML
    • e.g. title="TNT(C<sub>6</sub>H<sub>2</sub>(NO<sub>2</sub>)<sub>3</sub>CH<sub>3</sub>)"

Using tooltips for disabled elements

If the attribute disabled is set to the element, it also disables a tooltip (example below).

  • button.btn.btn-primary Attributes
    • Attribute type="button"
    • Attribute data-toggle="tooltip"
    • Attribute title="Disabled tooltip"
    • Attribute disabled (also disables a tooltip)
    • Text Disabled button

Example (does not show tooltip on hover)

To enable a tooltip for a disabled element, add a wrapping parent and set tooltip to it.

  • Wrapping parent span Attributes
    • Utility class .d-inline-block (set to the same display setting as button )
    • Attribute data-toggle="tooltip"
    • Attribute title="Disabled tooltip"
    • Target element button.btn.btn-primary Attributes
      • Attribute type="button"
      • Attribute style="pointer-events: none;" (to prevent receiving pointer events)
      • Attribute disabled
      • Text Disabled button
