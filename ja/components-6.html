インプットグループ

入力要素(input , textarea , etc.)の前後にテキスト、ボタンなどの補助機能を配置するための拡張用コンポーネント。

基本例

前置の場合
@
構造
  • グループ外枠 div.input-group
    • 前置グループ div.input-group-prepend
        span.input-group-text 属性
        • (アクセシビリティ)属性 id="basic-addon1"
        • テキスト @
    • フォームコントロール input.form-control 属性
      • 属性 type="text"
      • 属性 placeholder="Username"
      • アクセシビリティ属性 aria-label="Username"
      • アクセシビリティ属性 aria-describedby="basic-addon1"

(公式ドキュメントより)使用上のガイドラインは次の通り。

  • インプットグループ中にフォームコントロールは1つのみ
    • これを守らないとBootstrap 4方式バリデーションを用いる場合に問題を生じる
  • ラベル label はインプットグループの外部に用いる
    • これを守らないとアクセシビリティ上の支障を生じる可能性がある
注: 厳密な規則ではなく、以下に示す公式ドキュメントの例もこれに従っていないものが多い。
(バージョン4.1の時点) バリデーションへの対応がやや不完全で、実行中は枠の角が丸くならない。下記のバリデーション設定例を参照。

バリデーション

この構成は入力コントロールは1つだけという規則に従っていないため、Bootstrap 4方式バリデーションで問題を生じる。ただし不正(invalid)時に共通のメッセージを表示するという仕様であれば矛盾は生じない。次の設定例を参照。

First and last name
Please input your name
構造
  • グループ外枠 属性
    • バリデーション状態表示のため .was-validated を追加
    • 前置グループ
    • フォームコントロール input.form-control (×2) 属性
      • 属性 required を追加
    • 不正時メッセージ div.invalid-response
      • テキスト Please input your name

ジャンボトロン

例1 (基本構成)

Hello, world!

This is a simple hero unit, a simple jumbotron-style component for calling extra attention to featured content or information.

It uses utility classes for typography and spacing to space content out within the larger container.

Learn more

構造
  • div.jumbotron
    • h1.display-4
      • タイトル Hello, world!
    • p.lead
      • テキスト This is a simple hero unit, ...
    • hr.my-4
    • p
      • テキスト It uses utility classes ...
    • p.lead
      • a.btn.btn-primary.btn-lg 属性
        • 属性 href="#"
        • アクセシビリティ属性 role="button"
        • テキスト Learn more
